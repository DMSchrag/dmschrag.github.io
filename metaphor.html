Programming Metaphor

During this weeks lecture 03.03.2026 we learned about scaffolding and metaphors.

Project Overview

In this project we had to come up with a metaphor for a concept in programming, I opted for variable declaration in python. I crafted a small paper box, this would be representing the variable/variable name and in it is the actual variable value. Since the value is stored in the variable.

Variables Explained Visually

On the outside of the box I wrote the variable name, here called object_color as example.

Variable Components

In the box is then the actual value of the variable, in this case it has the value 3. This is to show that the variable name is just a label for the value it holds, and that the value can be changed while the variable name remains the same.
